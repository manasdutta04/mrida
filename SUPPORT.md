# Support

## Where to Ask for Help

- Usage/setup questions: open a GitHub Discussion (recommended)
- Bugs: open a GitHub Issue using the bug template
- Feature ideas: open a GitHub Issue using the feature template

## What to Include

- OS and tool versions
- steps to reproduce
- expected vs actual behavior
- logs/screenshots where possible

## Maintainer Contact

If Discussions are not enabled yet, open an issue titled:
`[Support] <short topic>`
